# generate: concept implementation

The following text describes a concept or set of concepts that the user would like to implement. 

- If a full specification is not provided, first generate the concept specification, otherwise faithfully implement from the given specification. 
- Generate a complete working implementation under `src/{name}/{name}Concept.ts` where `{name}` refers to the name of the concept.
- Generate a test file under `src/{name}/{name}Concept.test.ts` where `{name}` refers to the name of the concept.
